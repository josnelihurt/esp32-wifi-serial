About This Project

ESP32 WiFi Serial Bridge

This lightweight WiFi-to-Serial bridge provides emergency console access and remote administration for embedded systems like Raspberry Pi and Orange Pi. When SSH connectivity fails or network issues arise in your homelab infrastructure, this ESP32-based solution bridges serial console access over WiFi and MQTT, offering a reliable fallback for console access and basic administration. It's perfect for managing headless embedded systems without physical access, integrating seamlessly into homelab deployments for out-of-band management.

Key Features

  • Serial Bridge: Connect ARM devices via UART serial port
  • SSH Server: Direct SSH access to serial console on port 22
  • USB Debug Port: Built-in USB serial for debugging
  • MQTT Integration: Send/receive serial data over MQTT topics
  • Web Interface: Configuration and real-time monitoring
  • AP Mode: Automatic access point for initial setup
  • OTA Updates: Over-the-air firmware updates
  • Serial Logging: Circular buffer for serial data

Technical Details

Hardware: ESP32-C3 SuperMini / DevKitM-1

Platform: ESP32-C3 RISC-V @ 160MHz

Memory: 320KB RAM, 4MB Flash

Framework: Arduino on PlatformIO

Connectivity: WiFi 2.4GHz, MQTT, HTTP

Serial Ports: USB (ttyS0) + UART GPIO 0/1 (ttyS1)
